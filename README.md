### Welcome to Material Markdown!
**Shortcuts**
- Load Sample Page: Ctrl+M
 - Mac: Cmd+4
- Open File: Ctrl+Q
	- Mac: Cmd+5
- Save File: Ctrl+Shift+1
	- Mac: Cmd+2
- Save As File: Ctrl+Shift+Y
	- Mac: Cmd+3
- Toggle Blockquote: Ctrl+'
- Toggle Bold: Ctrl+B
- Toggle Italic: Ctrl+I
- Draw Link: Ctrl+K
- Toggle Unordered List: Ctrl+L
