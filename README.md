# Welcome to Material Markdown!
This is a simple and clean markdown editor with side by side preview.

**Shortcuts**
- Load Sample Page: Ctrl+P
- Open File: Ctrl+O
- Save File: Ctrl+S
- Toggle Blockquote: Ctrl+'
- Toggle Bold: Ctrl+B
- Toggle Italic: Ctrl+I
- Draw Link: Ctrl+K
- Toggle Unordered List: Ctrl+L
